The ADDIE Model

The ultimate learning designer guide

Analysis

This is the first step of the ADDIE model and relates to the analysis of the learners' needs and the goals to be reached.

Design

This phase is meant to set the learning guidelines: stages, tools, assessment instruments, exercises, content, subject matter analysis, lesson planning and media selection.

Let's go

Development

It is the step when instructional designers and developers create and assemble the content assets that were blueprinted in the design phase.

Let's go

Implementation

During the implementation phase, a procedure for trainers and learners is developed: it includes a review of the course curriculum as well as the teacinh methods.

Let's go

Evaluation

This last phase is used to review the training, get feedback, and make adjustments if necessary. The ADDIE Model is a continuous circle with overlapping boundaries.
